### SA-Faroo
![Faroo-logo](static/AppLogo.png)

## Description 
Adds the `faroo` search command to Splunk. This queries the Faroo.org API for web searches similar to google. It is limited to 10 queries per call.

## Installation 
* Grab a Faroo API key from <http://www.faroo.com/hp/api/api.html#key>
* Setup the app with the API key
* Start searching
